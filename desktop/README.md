# 照片打印后台桌面端

运行在打印店电脑上：拉取任务、按订单号下载照片、本地检索、人工确认后打印。

## 功能

- **自动下载**：定时请求后端，将待打印任务的照片按订单号下载到本地 `tasks/<订单号>/`。
- **后台页面**：浏览器打开 `http://localhost:3980`，按订单号/时间/状态检索，查看是否传完，点击「打开文件夹」或「开始打印」。
- **开始打印**：点击后标记为已打印并打开该订单的本地文件夹，由人工在系统中选择对应尺寸的打印机打印（或对接下方自动打印方案）。

## 配置

1. 复制 `config.example.json` 为 `config.json`。
2. 修改 `apiBase` 为你的后端地址（与小程序一致）。
3. `localBase` 为本地任务根目录（相对本项目），照片会下载到 `localBase/<订单号>/`。
4. `printers` 为预设的尺寸与打印机名称，用于后续扩展「按尺寸自动选择打印机」。

## 运行

```bash
npm install
npm start
```

然后访问 http://localhost:3980。如需仅执行一次下载，可运行：

```bash
node src/downloader.js
```

## 按尺寸分打印机自动打印（扩展）

当前「开始打印」会打开订单文件夹。若需按尺寸自动发送到不同打印机，可：

- **Windows**：使用系统默认「打印」或第三方命令行工具（如将图片转为 PDF 后调用指定打印机）。
- 在 `main.js` 的 `/print` 逻辑中，根据 `config.printers` 遍历订单内照片，按尺寸调用系统打印或外部脚本，将每张照片发送到对应 `name` 的打印机。
