# 照片打印微信小程序

## 流程

1. 用户通过**门店二维码**进入小程序（将小程序码或带 scene 的二维码贴在店内，扫码即打开本小程序）。
2. 输入购买时获得的**订单号**，点击「下一步」。
3. 系统校验订单并显示可上传张数，进入上传页。
4. 选择照片（数量不可超过订单允许张数），提交后等待打印。

## 配置

- 在 `app.js` 的 `globalData.apiBase` 中填写后端 API 的完整地址（需 https）。
- 微信公众平台 → 开发 → 开发管理 → 开发设置 → 服务器域名：在 request 合法域名中填入上述 API 域名。

## 开发

用微信开发者工具打开本目录，填入 AppID 后即可预览和上传。
