# 微信小程序 · 广告位管理

## 项目说明

本目录为微信小程序项目，聚焦**门店广告位**管理：多角色入口、看板、列表与门店维度的类型展示。  
当前数据为**静态演示**，用于预览 UI 与流程，**尚未对接后端接口**。

## 页面一览

| 路径 | 说明 |
|------|------|
| `pages/index` | 角色选择 |
| `pages/admin` | 管理者总览：看板、按类型统计、门店分类型表格 |
| `pages/slots` | 广告位列表；支持从看板带筛选参数；非「全部」时按门店+类型分组 |
| `pages/store-slots` | 某门店下按广告位类型展示缩略图（已投放 / 空置 / 即将到期） |
| `pages/advertiser` | 广告主：投放与安装照片 |
| `pages/inspector` | 巡店员：任务与打卡拍照 |

## 开发

用微信开发者工具**打开本目录**（`miniprogram`）为项目根目录，填入 AppID 后即可预览；对接后端后，在微信公众平台配置 request / uploadFile 等合法域名。
