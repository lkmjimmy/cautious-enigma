# 照片打印小程序系统

基于微信小程序的照片打印完整解决方案：用户扫码填订单号上传照片，后台自动下载、排版，人工审核后按尺寸分打印机打印。

## 系统架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────────────────┐
│  微信小程序      │────▶│  后端 API 服务   │────▶│  后台桌面端 (打印店电脑)   │
│  扫码/订单号    │     │  订单校验/存储   │     │  自动下载→排版→人工打印    │
│  上传照片       │     │  文件存储        │     │  多尺寸打印机               │
└─────────────────┘     └──────────────────┘     └─────────────────────────────┘
```

## 目录说明

| 目录 | 说明 |
|------|------|
| `backend/` | Node.js 后端 API：订单校验、SKU 限制、上传接口、任务列表 |
| `miniprogram/` | 微信小程序：扫码页、订单号输入、照片上传（数量受 SKU 限制） |
| `desktop/` | 后台桌面端：自动拉取任务、本地下载、自动排版、订单检索、一键打印 |

## 业务流程

1. **用户端**：扫描门店二维码 → 输入购买订单号 → 系统校验订单并显示可上传张数 → 上传照片并提交
2. **后端**：校验订单号、根据 SKU 限制上传数量、保存照片与任务
3. **后台电脑**：定时/实时拉取新任务 → 按订单号下载到本地 → 自动排版（以订单号为任务名）→ 人工按时间/订单检索 → 确认照片齐全后点击「开始打印」→ 按尺寸分发到预设打印机

## 本地运行

### 后端

```bash
cd backend && npm install && npm run dev
```

默认 `http://localhost:3000`，需配置数据库与上传目录（见 `backend/README.md`）。

### 微信小程序

用微信开发者工具打开 `miniprogram/`，填写 AppID，将「 request 合法域名」设为你的后端域名。

### 后台桌面端

```bash
cd desktop && npm install && npm start
```

首次运行需在设置中配置：后端 API 地址、上传/排版/打印输出目录、各尺寸对应的打印机名称。

## 环境要求

- Node.js 18+
- 微信开发者工具
- 后台电脑：Windows/macOS，已安装多台打印机并设置好不同尺寸

## 配置要点

- **SKU 与张数**：在后端或数据库中维护「商品 SKU → 可打印张数」映射，订单校验时根据订单商品取最大可上传张数。
- **打印机预设**：在桌面端配置如「6寸」「5寸」「证件照」等尺寸与打印机名称的对应关系，打印时按照片尺寸自动选择打印机。
